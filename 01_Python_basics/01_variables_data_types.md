<b>Casting</b>
```python
x = str(3)    # x will be '3'
y = int(3)    # y will be 3
z = float(3)  # z will be 3.0
```
<b>Get the Type</b>    
get the data type of a variable with the type() function.
```python
x = 5
y = "John"
print(type(x))
print(type(y))
```
